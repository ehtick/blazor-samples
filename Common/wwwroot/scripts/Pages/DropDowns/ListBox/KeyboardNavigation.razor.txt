<ActionDescription>
 <p>
    This example demonstrates keyboard interaction capabilities of the ListBox component for efficient, keyboard-driven navigation and selection. The documented shortcuts cover focus management, single-item navigation, and multi-selection patterns commonly used in desktop and accessible web interfaces.
  </p>

  <p><strong>Focus navigation</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Alt</kbd> + <kbd>J</kbd></span> — Focuses the first ListBox instance on the page.</li>
  </ul>

  <p><strong>Item navigation</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Up arrow</kbd></span> — Move focus to the previous option.</li>
    <li><span class="key-class"><kbd>Down arrow</kbd></span> — Move focus to the next option.</li>
    <li><span class="key-class"><kbd>Home</kbd></span> — Move focus to the first option.</li>
    <li><span class="key-class"><kbd>End</kbd></span> — Move focus to the last option.</li>
    <li><span class="key-class"><kbd>Space</kbd></span> — Toggle the selection state of the focused option.</li>
  </ul>

  <p><strong>Multi-selection</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>A</kbd></span> — Select all options.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Home</kbd></span> — Select the focused option and all options up to the first.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>End</kbd></span> — Select the focused option and all options down to the last.</li>
    <li><span class="key-class"><kbd>Ctrl</kbd> + <kbd>Arrow keys</kbd></span> — Use Ctrl with Up/Down arrows (or mouse interactions) to extend or modify multiple selections incrementally.</li>
  </ul>

 <p>
    Implement these shortcuts alongside proper focus management and ARIA attributes to ensure the ListBox remains accessible to keyboard users and assistive technologies while delivering efficient selection workflows.
 </p>
</ActionDescription>
