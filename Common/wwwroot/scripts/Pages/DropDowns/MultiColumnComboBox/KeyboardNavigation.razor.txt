<ActionDescription>
  <p>
    This example demonstrates the virtualization and comprehensive keyboard navigation capabilities of the
    <a href="https://www.syncfusion.com/blazor-components/blazor-multicolumn-combobox" aria-label="Navigate to Feature Tour of Blazor MultiColumn ComboBox" target="_blank" rel="noopener">Blazor MultiColumn ComboBox</a>,
    delivering optimized rendering for large datasets while ensuring accessible, efficient interaction.
  </p>

  <p>Virtualization highlights:</p>
  <ul>
    <li>Load and render only visible items to minimize DOM complexity and improve render times.</li>
    <li>Efficient handling of extensive datasets to preserve responsiveness during scrolling and filtering.</li>
    <li>Smooth scrolling experience without performance degradation on large lists.</li>
    <li>Reduced memory usage and predictable interaction latency for fast user feedback.</li>
  </ul>

  <p>Practical benefits include improved throughput for large collections, lower client memory consumption, and a consistently responsive user experience during typing, navigation, and selection.</p>

  <p>Keyboard navigation enables accessible operation and quick workflows. Supported shortcuts:</p>

  <p><strong>Focus Navigation</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Alt</kbd> + <kbd>J</kbd></span> – Focuses the first component instance.</li>
  </ul>

  <p><strong>Input Navigation</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Alt</kbd> + <kbd>Down arrow</kbd></span> – Opens the popup list.</li>
    <li><span class="key-class"><kbd>Alt</kbd> + <kbd>Up arrow</kbd></span> – Closes the popup list.</li>
    <li><span class="key-class"><kbd>Home</kbd></span> – Moves the cursor to the start of the input.</li>
    <li><span class="key-class"><kbd>End</kbd></span> – Moves the cursor to the end of the input.</li>
    <li><span class="key-class"><kbd>Tab</kbd></span> – Moves focus to the next tab-index element.</li>
    <li><span class="key-class"><kbd>Shift</kbd> + <kbd>Tab</kbd></span> – Moves focus to the previous tab-index element.</li>
  </ul>

  <p><strong>Selection</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Enter</kbd></span> – Selects the focused item and closes the popup.</li>
  </ul>

  <p><strong>Popup Navigation</strong></p>
  <ul>
    <li><span class="key-class"><kbd>Down arrow</kbd></span> – Selects the first or next item.</li>
    <li><span class="key-class"><kbd>Up arrow</kbd></span> – Selects the previous item.</li>
    <li><span class="key-class"><kbd>Esc</kbd></span> – Closes the popup list.</li>
  </ul>

  <p>
    For implementation guidance and examples, see the component documentation: 
    <a href="https://blazor.syncfusion.com/documentation/multicolumn-combobox/getting-started/" target="_blank" rel="noopener" aria-label="Navigate to Blazor MultiColumn ComboBox Getting Started documentation">Getting Started with Blazor MultiColumn ComboBox</a>.
  </p>
</ActionDescription>
